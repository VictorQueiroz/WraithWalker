# Native Host

Native-messaging host for the browser extension when you need OS-level folder
reveal, scenario actions, or command-based editor integrations without the
local server handling those actions.

## Do you need this?

Usually, no. Most workflows can use `wraithwalker serve` and the extension
without installing the native host.
