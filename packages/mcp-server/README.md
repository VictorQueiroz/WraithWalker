# MCP Server

An MCP server that exposes WraithWalker fixture roots and browser-backed trace
state to AI agents.

## Quick start

Most users should start this through the main CLI:

```bash
wraithwalker serve
```

Use this package directly only when you want the standalone
`wraithwalker-mcp` binary or the programmatic server APIs.
