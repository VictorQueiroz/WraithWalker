# CLI

The `wraithwalker` CLI manages fixture roots, capture config, HAR/import flows,
editor context generation, scenarios, and the local combined MCP+tRPC server.

## Quick start

```bash
npm install -g @wraithwalker/cli
wraithwalker serve
```
